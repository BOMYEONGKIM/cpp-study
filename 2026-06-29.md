## 문제1 - 문자열 섞기
```cpp
#include <string>
#include <vector>

using namespace std;

string solution(string str1, string str2) {
        string answer = "";

    int index1=0,index2=0;
    
    for(int i=0;i<str1.length()+str2.length();i++)
    {
        if(i%2==0)
            answer+=str1[index1++];
        else
        answer+=str2[index2++];

    }
    return answer;
}
```
### 배운 것
- str1과 str2의 길이가 항상 같으므로 굳이 분기를 안둬도 됨.
- `push_back()` : 문자열/벡터 맨 뒤에 원소 추가
- `+=` 와 `push_back` 은 string에서 동일한 동작
 ```
#include <string>
#include <vector>

using namespace std;

string solution(string str1, string str2) {
    string answer = "";
    for(int nIndex=0; nIndex<str1.size(); nIndex++)
    {
        answer += str1[nIndex];
        answer += str2[nIndex];
    }
    return answer;
}
```

## 문제2 - 문자리스트를 문자열로 변환하기
```cpp
#include <string>
#include <vector>

using namespace std;

string solution(vector<string> arr) {
    string answer = "";
    for(int i=0;i<arr.size();i++)
    {
    answer+=arr[i];
    }
    return answer;
}
```
### 배운 것
- `push_back()` 의 인자로는 'char'형만 가능
- `+=` : `char`도 되고 `string`도 됨


## 문제3 - 문자열 곱하기
```cpp
#include <string>
#include <vector>

using namespace std;

string solution(string my_string, int k) {
    string answer = "";
    for(int i=0;i<k;i++)
    {
        answer += my_string;
    }
    return answer;
}
```
### 배운 것
- `append(n, c)` : 문자 하나를 n번 반복 추가, `string` 반복은 불가

## 문제4 - 더 크게 합치기
```cpp
#include <string>
#include <vector>

using namespace std;

int solution(int a, int b) {
    int answer = 0;
    string result1,result2;
    
    result1 = to_string(a) + to_string(b);
    result2 = to_string(b)+ to_string(a);
    
    if(stoi(result1)>stoi(result2))
        answer=stoi(result1);
    else
        answer=stoi(result2);
    
    return answer;
}
```
### 배운 것
- `max(a, b)` : 두 값 중 큰 값 반환, if-else 대신 사용 가능 (`<algorithm>` 필요)

## 문제5 - 두 수의 연산값 비교하기 
```cpp
#include <string>
#include <vector>

using namespace std;

int solution(int a, int b) {
    int answer = 0;
    answer = max(stoi(to_string(a)+to_string(b)),2*a*b);
    return answer;
}
```
### 배운 것 
- max는 같으면 첫 번째 인자를 반환
