## 오늘 푼 문제
2개 이상의 변수 혹은 문자열 출력하기

## 코드
```cpp
#include <iostream>

using namespace std;

int main(void) {
    int a;
    int b;
    cin >> a >> b;
    cout << "a = " << a << endl;
    cout << "b = " << b << endl;

    return 0;
}
```

## 배운 것
- 문자열과 변수를 이어서 출력할 때 `<<` 로 연결할 수 있다
- `cout << "a = " << a << endl;`
