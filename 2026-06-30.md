## 문제1 - n의 배수
### 코드
```cpp
#include <string>
#include <vector>

using namespace std;

int solution(int num, int n) {
    return ((num%n)==0) ? 1:0;
}
```
### 배운 점
- 3항 연산자 : `조건 ? 참일때값 : 거짓일때값`, if-else를 한 줄로 줄일 수 있음
  ```
  return num % n == 0;
  ```
- `bool` 값은 자동으로 `int`로 변환됨 (`true` → 1, `false` → 0)
- 조건식 자체가 이미 0/1 값이면 3항 연산자로 다시 변환할 필요 없음

## 문제2 - 공배수 
### 코드
```cpp
#include <string>
#include <vector>

using namespace std;

int solution(int number, int n, int m) {
    return (number % n == 0 && number % m == 0);
}
```
### 배운 점
- "나머지가 같다"는 것과 "둘 다 배수다"는 다른 의미
  - `number % n == number % m` ≠ `number % n == 0 && number % m == 0`
  - 반례: `number=1, n=2, m=2` → 나머지는 같지만(1==1) 둘 다 배수는 아님(1은 2의 배수 아님)
  - `n == m` 인 경우 항상 나머지가 같아서 틀린 조건이 true가 되어버림
- 배수 조건은 `number % n == 0` 형태로 직접 0인지 확인해야 함
- 직관적으로 맞아 보이는 조건도 반례를 찾아서 검증하는 습관 필요

## 문제3 - 홀짝에 따라 다른 값 반환하기
### 코드
```cpp
#include <string>
#include <vector>

using namespace std;

int solution(int n) {
    int answer = 0;
    if(n%2==0)//짝수
    {
        for(int i=1;i<=n;i++)
        {
            answer += (i%2==0) ? i*i : 0;
        }
    }
    else
    {
                for(int i=1;i<=n;i++)
        {
            answer += (i%2==1) ? i : 0;
        }
    }
    return answer;
}
```
### 배운 점
- n에서 직접 2씩 감소시키며 처리하면 짝수/홀수 분기 없이 반복문 하나로 처리 가능
- 비트연산으로 짝수/홀수 판별 및 나누기 가능
  - `n & 1` : `n % 2`와 동일, 홀수면 1
  - `(n+1) >> 1` : `(n+1)/2`와 동일
- 1~n 홀수 합 = `k²` (k = 홀수 개수), 짝수 제곱 합 = `2k(k+1)(2k+1)/3` 공식으로 O(1) 계산 가능
- 현업에서는 가독성이 성능보다 우선
  - 비트연산/압축된 수학 공식보다 명확한 반복문이 유지보수에 유리
  - 임베디드, 실시간 시스템처럼 자원 제약이 큰 경우는 최적화 우선 고려
- 코딩테스트에서는 시간복잡도 최적화 연습도 의미 있음, 다만 실무 코드는 가독성 중심으로 작성

##문제4 - 조건 문자열
###코드
```cpp
  #include <string>
#include <vector>

using namespace std;

int solution(string ineq, string eq, int n, int m) {
    if (ineq == ">") {
        if (eq == "=") return n >= m;
        else return n > m;
    } else {
        if (eq == "=") return n <= m;
        else return n < m;
    }
}
```
### 배운 점
- `switch`는 `string` 타입에 사용 불가, `int`나 `char`만 가능
- 문자열 비교는 `if-else` + `==` 로 처리해야 함
- `string` 비교 시 작은따옴표(`'='`)가 아닌 큰따옴표(`"="`)를 써야 함
- `bool` 값을 그대로 `return` 하면 `true`/`false`가 자동으로 `1`/`0`으로 변환됨

## 문제 5 - flag에 따라 다른 값 반환하기 
### 코드
```cpp
#include <string>
#include <vector>

using namespace std;

int solution(int a, int b, bool flag) {
    if(flag) return a+b;
    else return a-b;
}
```
